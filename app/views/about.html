About Cistrome DB resources

Cistrome DB is a comprehensive resource of hg38 and mm10 ChIP-seq data collection. Here is a brief introduction about the workflow.

  • Mapping: bwa and post-processing with MAPQ 1
  • peak calling: latest macs2 after removing duplicate reads.
  • Peak conservation analysis
  • Peak meta region analysis
  • Peak regulatory potential analysis
  • ChIP-seq quality control analysis

Cistrome DB function

The main cistrome performs data search function with 4 views and 1 inspector:

  • Quality control view
  • Motif ScanView
  • Regulatory Potential View
  • Similar ChIP-seq dataset view

    • CistromeDB Pulication Policies and Guidelines

    The Cistrome DB program has established the following policy to clarify freedom to publish or present on findings using Cistrome DB data. Data users may freely download and analyze CistromeDB data with the following restrictions:

    1. We are now working on a global integrative analysis study based on Cistrome DB.

    2. Users are not allowed to download more than 100 datasets.

    3. Data users should properly cite the Cistrome Project in any publication, please use the following two publications temporarily:.

      1. Sun H, Qin B, Liu T, Wang Q, Liu J, et al. (2013) Data and Text Mining CistromeFinder for ChIP-seq and DNase-seq data reuse. Bioinformatics: 1–3.
      2. Qin B, Zhou M, Ge Y, Taing L, Liu T, et al. (2012) CistromeMap: a knowledgebase and web server for ChIP-Seq and DNase-Seq studies in mouse and human. Bioinformatics 28: 1411–1412.
